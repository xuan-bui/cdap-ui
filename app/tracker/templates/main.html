Search Tips
Search datasets by
The name - Input the dataset name by using the format entity-name:<dataset-name> (example: entity-name:userprofiles)
A field in the dataset schema - Input the field name of a schema using the format schema:<field-name> (example: schema:zipcode)
A property annotated on the dataset - Input a property as a query, using the format <key>:<value> (example: env:production)
A tag annotated on the dataset - Input a name of a tag as a query, using the format tags:<tag-name> (example: tags:batch)
Wild card - Append * to the end of the search term (example: user* or tags:b*)
